Implement a solution to the Wolves and Chicks Puzzle from the game Professor Layton and the Curious Village.

Get the three wolves and three chicks seen below to the other side of the river while obeying the following conditions.

No more than two animals can ride the raft at the same time.

There must be at least one animal on the raft in order for it to move.

If more wolves than chicks stay on either side of the river, the wolves will eat the chicks, and you'll have to start over.

You can move the raft as many times as you like, but this feat can be accomplished in as few as 11 moves.

def chickenWolvesSuccessors( state, hist = None ):
    raftIsLeft, wolvesLeft, chicksLeft, wolvesRight, chicksRight = state 
    children = []
    if raftIsLeft:
      # Move 1 or 2 Wolf
      for n in range(1,3):
        if wolvesLeft >= n:
            ns = [ False, wolvesLeft - n, chicksLeft, wolvesRight + n, chicksRight ]
            if isValid( ns ):
                children.append( ns )
      # Move 1 or 2 chickens
      for n in range(1,3):
        if chicksLeft >= n:
            ns = [ False, wolvesLeft, chicksLeft - n, wolvesRight, chicksRight + n ]
            if isValid( ns ):
                children.append( ns )
      # Move 1 wolf and 1 chicken
      if chicksLeft >= 1 and wolvesLeft >= 1:
          ns = [ False, wolvesLeft - 1, chicksLeft - 1, wolvesRight + 1, chicksRight + 1 ]
          if isValid( ns ):
              children.append( ns )
    else:
      # Move 1 or 2 Wolf
      for n in range(1,3):
        if wolvesRight >= n:
            ns = [ True, wolvesLeft + n, chicksLeft, wolvesRight - n, chicksRight ]
            if isValid( ns ):
                children.append( ns )
      # Move 1 or 2 chickens
      for n in range(1,3):
        if chicksRight >= n:
            ns = [ True, wolvesLeft, chicksLeft + n, wolvesRight, chicksRight - n ]
            if isValid( ns ):
                children.append( ns )
      # Move 1 wolf and 1 chicken
      if chicksRight >= 1 and wolvesRight >= 1:
          ns = [ True, wolvesLeft + 1, chicksLeft + 1, wolvesRight - 1, chicksRight - 1 ]
          if isValid( ns ):
              children.append( ns )
    return children

Implementation of Depth First Search

We will now implement the depth first search algorithm to solve the puzzle.

I will use an implementation using a queue instead of a recursive solution for reasons that will become apparent later

def search( initial, successorFunc, limit = 1000 ):
    queue = [ ( initial, [initial] ) ]
    numNodes = 1
    sol = None
    while( len( queue ) > 0 ):
        # current, chist = queue[-1]
        # queue = queue[0:-1]
        current, chist = queue.pop()
        if (current[1] == 0) and (current[2] == 0):
            sol = (numNodes, chist)
            break
        #print("current state", current, chist )
        #print("queue", queue)
        children = successorFunc( current )
        #print("children", children)
        for c in children:
            if c not in chist:
                queue.append((c, chist + [c] ))
                numNodes = numNodes + 1
                if ( numNodes > limit ):
                    return None
    return sol
